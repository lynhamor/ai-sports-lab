# ai-sports-lab

# Sports Simulation Engine

Version: **1.0.1**

A simple backend project for a **sports betting simulation engine** using **Spring Boot** and **Spring AI (Ollama)**.

This project generates simulated sports data such as teams, matches, schedules, and stats using AI, while the core game logic and betting rules are handled by application code.

## Goal

To build a simulation system that can:

* Generate sports events using AI
* Create teams and match schedules
* Simulate basic sports statistics
* Provide a foundation for a betting simulation engine

## Tech Stack

* Java 23
* Spring Boot 3.2.5
* Spring AI (Ollama integration)
* Spring Data JPA
* MariaDB
* Maven

## Current Features (v1.0.1)

* Basic Spring Boot setup
* Ollama AI integration (initial setup)
* Project structure for simulation engine
* Database configuration (MariaDB ready)

## Planned Features

* AI-generated teams and leagues
* Match schedule generation
* Match simulation engine
* Betting logic (odds, bets, settlement, etc)
* Wallet system
* REST API for simulation control

## Notes

This is an early-stage project. Core simulation and betting logic are still under development.
